Use-Case Specification: Informationen aus einer Quelle laden

1.                  Informationen laden

1.1               Kurzbeschreibung

Der Use-Case "Informationen aus einer Quelle laden" gibt dem Benutzer die Möglichkeit Informationen über Medien zu bearbeiten. Dabei wird Zeit bei der Eingabe der Beschreibung von den Medien gespart, somit muss der Benutzer sich keinen eigenen Text ausdenken und diese noch selbst eintippen.

2.                  Flow of Events

2.1               Basic Flow

2.2               Alternative Flows

n.a.

3.                  Spezifische Anforderungen

3.1               Medien

Der Benutzer muss die Medien, die er anbietet auch wirklich besitzen. Auch ist das Wissen des Inhalts wichtig, damit dieser die richtige Information auswählen kann.

3.2               Internetverbindung

Um die Informationen laden zu können, muss eine Internetverbindung bestehen.

 

4.                  Voraussetzungen

Der Benutzer muss angemeldet sein und den Inhalt der Medien wissen.

5.                  Nachbedingungen

Wenn der Benutzer die Daten bestätigt hat, müssen diese in der Datenbank gespeichert sein.
