Use-Case Specification: Ausleihen

1.                  Ausleihen

1.1               Kurzbeschreibung

Der Use-Case "Ausleihen" gibt dem Benutzer die Möglichkeit Medien von anderen Benutzern auszuleihen. Gleichzeitig können sie ihre Medien auch zum Verleih anbieten. Beim Ausleihen wird ein Fomular abgeschickt, welche die Medien beinhaltet, die ausgeliehen werden sollen. Nachdem das Formular abgeschickt wurde, soll der Benutzer benachrichtigt, dessen Medien ausgewählt wurden.

2.                  Flow of Events

2.1               Basic Flow

2.2               Alternative Flows

n.a.

3.                  Spezifische Anforderungen

3.1               Medien

Der Benutzer muss die Medien, die er anbietet auch wirklich besitzen.

3.2               Internetverbindung

Um die Medien auszuleihen, muss eine Internetverbindung bestehen.

 

4.                  Voraussetzungen

Der Benutzer muss angemeldet sein und die ausgewählten Medien müssen "ausleihbar" gekennzeichnet sein.

5.                  Nachbedingungen

Die Benachrichtigung an den Benutzer, dessen Medien ausgeliehen werden sollen, muss beim Benutzer eingehen. Auch ist es wichtig, dass der Status der ausgeliehenen Medien auf "ausgeliehen" wechselt. Ebenfalls ist ein Eintrag vorhanden, der beinhaltet welcher Benutzer bei welchem Benutzer die Medien ausgeliehen hat und welche Medien es genau sind.

6.                  Anlagen

[Extension points of the use case.]

6.1               Narrative(Cucumber)

	
	Feature: Benutzer können Titel von anderen Benutzern ausleihen
	  Stehen mehrere Exemplare zur Auswahl und sind verfügbar, kann der Benutzer auswählen, von wem er leihen möchte

	Szenario: Nutzer leiht Titel aus
	Gegeben Benutzer ist eingeloggt und mindestens ein Exemplar des Titels ist verfügbar
	Wenn Benutzer auf ausleihen klickt
	Dann bekommt der Besitzer des Exemplars eine Benachrichtigung

	Szenario: Nutzer leiht Titel aus
	Gegeben Benutzer ist eingeloogt und kein Exemplar des Titels ist verfügbar
	Wenn Benutzer auf ausleihen klickt
	Dann bekommt bekommt er angezeigt, dass der Titel nicht verfügbar ist
